Setting up your Daily View screen

Two short steps and today’s plan will be up on the wall — no technical experience needed.

  1. 1

    Set up the viewer screen

    On the tablet or display itself:

    1. Plug it in and connect it to your home Wi‑Fi, as you would with any tablet.
    2. Open a web browser and go to dailyview.org/display
    3. Add it to the home screen (share/menu button → Add to Home Screen), so it reopens automatically if the screen restarts.
  2. 2

    Pair the two together

    Back on your own phone or computer:

    1. Sign in to your Daily View dashboard, then go to Devices → Add device and give it a name (e.g. “Mum’s kitchen screen”).
    2. You’ll see a short code and a QR code — it’s valid for 15 minutes.
    3. Back on the screen, type the code into the box shown there and select Connect.

That’s it — the screen updates itself automatically from here. Find it a stable, plugged-in spot where it’s easy to glance at. The person using it never needs to touch a setting or log in.
